# C语言 冒泡排序

## 运行代码

```C
#include <stdio.h>
#include <math.h>

int main() {
	
	int array[10] = {3, 6, 7, 1, 2, 0, 8, 4, 5, 9};

	for (int j = 0; j < 9; j++)	{
		for (int i = 0; i < 9; i++) {
			if (array[i] < array[i+1])
			{
				array[i] = array[i] + array[i+1];
				array[i+1] = array[i] - array[i+1];
				array[i] = array[i] - array[i+1];
			}
		}
		for (int i = 0; i < 10; i++) {
			printf("%d\t", array[i]);
		}
		printf("\n");
	}

}
```
