# 如何在手机上运行命令? 

## 问题 (仅限安卓)

手机无法运行命令行或者bash, 也无法使用 GitHub 同步本地文件夹. 

## 解决方案

Termux 终端 允许手机运行命令行, 并且可以运行 Git 指令. 

Termux 官网 下载链接: [Termux](https://termux.dev/cn/index.html)
